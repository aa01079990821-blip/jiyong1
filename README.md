# 오토워크룸 MVP

직장인을 위한 AI 업무 자동화 자료실의 첫 화면 MVP입니다. 정적 HTML/CSS/JS로 구성되어 있으며, 무료 콘텐츠, 프리미엄 멤버십, 광고 슬롯, 제휴 추천 도구, 뉴스레터 CTA가 한 화면에서 자연스럽게 보이도록 만들었습니다.

## 실행 방법

1. 이 폴더에서 간단한 로컬 서버를 실행합니다.

```bash
python3 -m http.server 8000
```

2. 브라우저에서 아래 주소를 엽니다.

```text
http://localhost:8000
```

## 포함된 기능

- 무료/프리미엄 콘텐츠 카드
- 검색과 카테고리 필터
- 멤버십 가격 UI
- 광고 자리 표시 영역
- 제휴/추천 도구 섹션
- 뉴스레터 이메일 수집 CTA
